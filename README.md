# vue-temp-js-pc

## 目录

```shell
├─api               # api请求
├─assets            # 静态资源文件
├─components        # 公共组件存放位置
├─directives        # 自定义指令
├─filters           # 自定义 过滤器
├─pages             # 子页面 文件夹
│  └─home           # home页面
│     └─components  # 存放home这个页面使用的一些组件
|     └─xxxxxxxxx   # home下的子页面
|     └─index.vue   # home页面   
├─router            # 路由配置
├─store             # vuex配置
```
